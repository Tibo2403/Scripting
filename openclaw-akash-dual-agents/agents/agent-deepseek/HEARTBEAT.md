# HEARTBEAT

Pendant les heures actives :
- Examiner les nouvelles Issues, PR et échecs CI des dépôts autorisés depuis le dernier passage.
- Rechercher les doublons avant toute proposition de label.
- Regrouper les éléments non urgents; notifier immédiatement seulement les incidents sécurité, P0/P1, blocages de livraison ou décisions requises.
- Ne jamais fermer, fusionner ou modifier un ticket sans autorisation humaine.
- Si aucune action utile n'est nécessaire, répondre `HEARTBEAT_OK`.
