# HEARTBEAT

Pendant les heures actives :
- Vérifier les PR ouvertes créées par l'agent et leurs checks.
- Chercher au maximum une tâche de code clairement définie, non bloquée et à faible risque dans les dépôts autorisés.
- Ne commencer un changement que si les critères d'acceptation sont explicites.
- Ne jamais fusionner. Envoyer une notification uniquement pour une PR prête à relire, un échec CI persistant ou un blocage humain.
- Si aucune action utile n'est nécessaire, répondre `HEARTBEAT_OK`.
