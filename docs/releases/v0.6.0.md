# Scripting Toolkit v0.6.0

## Highlights

This pre-1.0 release presents Scripting as a production-minded automation toolkit rather than an unstructured script collection.

- Clear five-minute quick start for PowerShell, Bash and Python users.
- Stronger project positioning around AI infrastructure, system administration, developer tooling and authorized security labs.
- Contributor guidance covering validation, safety and sensitive-data handling.
- Existing CI coverage for PowerShell, Bash and Python documented prominently.
- Discoverability topics proposed for the GitHub repository settings.

## Included capabilities

- Codex workspace auditing and validation.
- Private Ollama and Open WebUI deployment helpers.
- Read-only MCP server for repository script discovery.
- LiteLLM and budget-aware model-routing experiments.
- Microsoft 365 and system-administration automation.
- Guarded security-lab workflows with authorization and dry-run controls.

## Validation

The repository CI validates PowerShell syntax and PSScriptAnalyzer findings, Bash syntax and ShellCheck findings, Python compilation and unit tests, plus dedicated safety smoke tests.

## Safety notice

Security-related scripts are intended only for environments where the operator has explicit authorization. Do not commit credentials, customer data, real targets, scan output or tenant identifiers.

## Upgrade notes

No breaking runtime change is introduced by the public-repository quality update. The main changes are documentation and contributor experience improvements.
