# Demo Media Plan

Store screenshots in `docs/media/`.

Recommended captures:

- `docs/media/ci-success.png` - GitHub Actions `Script Validation` success.
- `docs/media/linux-safety-tests.png` - local safety test output.
- `docs/media/compatibility-matrix.png` - compatibility matrix preview.
- `docs/media/dry-run.gif` - short demo of a sensitive script dry-run.

Do not capture real target lists, credentials, scan output, packet captures, or customer data.
