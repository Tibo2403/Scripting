# Portfolio Brief

## Problem

Script collections can become risky when they lack validation, safety guards, and clear usage boundaries. This repository organizes administration and security scripts with CI and explicit authorization checks.

## What This Shows

- PowerShell and Bash automation.
- GitHub Actions validation for script quality.
- Safety design for high-risk scripts.
- Documentation for compatibility, examples, and contribution workflows.
- Security-conscious repository hygiene.

## Demo Flow

1. Run `bash scripts/tests/test-linux-safety.sh`.
2. Run `scripts/powershell/Test-ScriptSyntax.ps1`.
3. Show the compatibility matrix.
4. Demonstrate `--dry-run --yes-i-am-authorized` on a Linux script.

## Recruiter Notes

This project demonstrates automation, systems administration, scripting quality, and security-aware engineering.
